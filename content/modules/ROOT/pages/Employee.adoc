=  Employee API

== GET

----
Path param: /$id

Description: Return a specific Employee by ID, or do nothing

Response:
 - firstname 
 - lastname 
 - email 
 - phone number
 - level (optional)
----

----
Path param: /

Description: Return the list of all (visible) Employees

Response: list of Employee as returned by GET /$id
----


== PUT

----
Path param: /$id

Description: Update an existing customer or do nothing

Params:
 - firstname (optional)
 - lastname (optional)
 - email (optional)
 - phone number (optional)
----

== POST

----
Path param: /

Description: Add a new Employee

Params:
 - id (optional)
 - firstname
 - lastname
 - email
 - phone number
----
