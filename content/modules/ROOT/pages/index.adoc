= {lab_name}: A Capture The Flag game

Welcome to the Code Red Breach Challenge!

== What is this challenge?

This lab is a Capture The Flag (CTF) game where you'll step into the role of a DevSecOps expert investigating a security breach.

The game is divided into missions and each mission ends when the mission flag is discovered.
A flag is an element of information related to security, such as a password, an IP address, or other sensitive data.

This CTF game has a quest of 4 missions set in sequence (the next mission unlocks only when the previous one is complete)
and some extra missions that can be done in parallel to collect more points.

== What you'll learn

In this challenge, you will:

* Investigate security vulnerabilities in an OpenShift environment
* Analyze compromised applications and CI/CD pipelines
* Use Red Hat Advanced Cluster Security to identify threats
* Understand OpenShift GitOps and pipeline security
* Apply DevSecOps principles to find and exploit security weaknesses
* Learn secure coding and infrastructure practices

== Who this is for

This challenge is designed for developers, security professionals, and platform engineers who want to:

* Practice DevSecOps skills in a safe environment
* Learn about common security vulnerabilities in cloud-native applications
* Understand how attackers exploit weaknesses in CI/CD pipelines
* Gain hands-on experience with Red Hat security tools

== Challenge environment

Your environment includes:

* OpenShift cluster with a compromised application
* CTF Dashboard for mission tracking and leaderboard
* Red Hat Advanced Cluster Security (ACS)
* OpenShift GitOps (ArgoCD)
* Tekton pipelines
* Gitea source code repository

== Winning the CTF

To win, you must collect a maximum number of points.
Each discovered flag brings you a certain amount of points:

Point Zero                  : 10 points

Code Red Quest
 - 1. Run !                 : 100 points

 - 2. Captain Hook          : 150 points

 - 3. Blind Test            : 80 points

 - 4. Pearl Harbour         : 200 points

 - 5. Inception             : 300 points

 - 6. I've got you babe     : 350 points

Side Missions
 - Grizzly Credentials      : 50 points

 - Chessmaster              : 50 points

 - Vault Tec                : 50 points

 - Moby-Dick                : 50 points

 - Hypotek                  : 50 points


In case of a tie, the fastest wins, so you also have to be fast!

To help you go faster, you can unlock mission's hints, but they will cost you some points.
