== Challenge breakdown

=== Main Quest (Sequential missions)

1. **Point Zero** (10 points) -- Initial reconnaissance
2. **Run!** (100 points) -- First exploitation
3. **Captain Hook** (150 points) -- Pipeline investigation
4. **Blind Test** (80 points) -- Hidden vulnerabilities
5. **Pearl Harbour** (200 points) -- Advanced exploitation
6. **Inception** (300 points) -- Deep dive attack
7. **I've got you babe** (350 points) -- Final challenge

=== Side Missions (Parallel missions)

* **Grizzly Credentials** (50 points)
* **Chessmaster** (50 points)
* **Vault Tec** (50 points)
* **Moby-Dick** (50 points)
* **Hypotek** (50 points)

== Prerequisites

This challenge assumes:

* Basic understanding of Kubernetes and OpenShift
* Familiarity with CI/CD concepts
* Basic command-line skills
* Understanding of security concepts (secrets, authentication, authorization)
* Problem-solving mindset

No prior CTF experience is required, but curiosity and persistence are essential!

== Getting started

Start by opening the OpenShift web console.
You'll land in the namespace of the application that has been hacked.
You can notice that a pipeline has run, which created the malicious application, but the hacker, somehow, managed to destroy the image before leaving.
This explains why the Pod is not there.

Pay attention to the PipelineRun object.

Good luck!
